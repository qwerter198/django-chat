# Django聊天室

了解如何使用 Django 和 Channels 建立即時聊天應用程式。

## 建置步驟

1. 安裝所需套件
    
    ```bash
    pip install -r requirements.txt
    ```
    
2. 測試web伺服器
    
    ```bash
    python manage.py runserver
    ```
    

建置完成即可進行聊天室體驗


若要正式將Django應用程序到生產環境，
通常涉及設置Web服務器（如Nginx或Apache）、數據庫服務器、應用服務器（如Gunicorn或uWSGI）以及配置環境變量等步驟。
